# Экспериментально-парвовой режим

**Экспериментальный правовой режим (ЭПР)** — это особый правовой инструмент, позволяющий на ограниченный срок и определенной территории отступить от общих требований законодательства для апробации и внедрения инновационных технологий.

## Документы

В рамках ЭПР разрабатывается и утверждается следующий пакет документов:

1. Документы для установления ЭПР:

    - **Инициативное предложение** — документ, с которым инициатор (юридическое лицо, ИП или ФОИВ) обращается для установления режима. Содержит описание инновации, предлагаемое специальное регулирование и меры по минимизации рисков.

    - **Программа ЭПР** — основной документ, утверждаемый постановлением Правительства РФ или актом Банка России. Программа ЭПР включает:

        - срок действия режима (обычно 3 года);

        - территорию проведения эксперимента;

        - перечень общих требований, которые не применяются к участникам;

        - специальные требования, которым должны соответствовать участники.

2. Внутренние документы участника ЭПР:

    Субъект ЭПР (участник) обязан разработать и утвердить для себя:

      - положение о внутреннем контроле качества и безопасности деятельности (если применимо);

      - правила проведения деятельности в рамках ЭПР (например, правила проведения телемедицинских консультаций);

      - формы реестров и отчетности для оценки эффективности режима.

3. Отчетные документы:

    Отчет о деятельности — участники обязаны регулярно отчитываться о ходе реализации ЭПР и достигнутых результатах.
